# LLM-Benchmarking-Framework

A scientific, reproducible framework for evaluating the performance of freely accessible Large Language Models.

## Project Goal
To develop a benchmarking suite that evaluates LLMs on 220+ original prompts across categories including reasoning, knowledge retrieval, and coding, ensuring objective performance metrics.

## Directory Structure
- `01_Proposal/`: Contains the formal project scope and supervisor approval.
- `02_Reports/`: Weekly progress reports (PPP method).
- `03_Code/`: Python automation and orchestration scripts.
- `04_Datasets/`: The 220-prompt library (JSON/CSV).
- `05_Logs_Results/`: Raw experiment data and analysis logs.
- `06_Final_Report/`: The 50-80 page research paper in progress.
